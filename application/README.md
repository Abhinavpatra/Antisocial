## Get started
0. Check linting
   bunx eslint 

1. Install dependencies

   bun install
   
react-native-reanimated@latest react-native-safe-area-context@latest
2. Start the app

   bunx expo start
For ios:   
   bunx expo start --tunnel 

3. Compiling

You can compile your app locally with the run commands:
```bash
   npx expo run:ios
   npx expo run:android
```
